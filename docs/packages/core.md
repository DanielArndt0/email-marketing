# Package Core

O pacote `core` concentra o núcleo da aplicação.

## Responsabilidades esperadas

- entidades de domínio
- value objects
- regras de negócio
- contratos internos
- casos de uso da aplicação
- portas/interfaces para abstrações

## Objetivo

Manter o coração do sistema desacoplado de:

- Fastify
- PostgreSQL
- Redis
- BullMQ
- Nodemailer
- SMTP
